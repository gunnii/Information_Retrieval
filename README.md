# Information Retrieval Term Project
### Korea Univ. 2016-2학기 COSE472

## 기간
    * 2016.09 ~ 2016.12

## 구현환경
    * Windows 10
    * Visual Studio 2015
    * C++ (Using STL, No extra library) 
    * Release x64

## 구현내용
    * Document Parsing
    * Stemming (using porter2 Stemmer)
    * Stopword Removing (불용어 제거)
    * Indexing (역색인)
    * Ranking ( Vector Space Model & Language Modelv & Smoothing)
    * Evaluation
